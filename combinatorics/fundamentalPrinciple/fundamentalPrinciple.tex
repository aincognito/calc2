\section{Fundamental Principle of Counting}

\begin{example}[example 1]
Pat has 3 distinct shirts and 2 distinct pairs of pants. Getting dressed requires one of each.  
In how many ways can Pat get dressed?\\
We make a tree diagram to determine the possible outfits Pat can create.
\begin{image}
\begin{tikzpicture}
\draw[mark=*,mark size=3pt,mark options={color=blue}] plot coordinates {(0,0)} node[above, blue]{Pat};
\draw (0,0) -- (3,4) ;
\draw[mark=*,mark size=3pt,mark options={color=blue}] plot coordinates {(3,4)} node[above, blue]{Shirt 1};
\draw (0,0) -- (3,0) ;
\draw[mark=*,mark size=3pt,mark options={color=blue}] plot coordinates {(3,0)} node[above, blue]{Shirt 2};
\draw (0,0) -- (3,-4) ;
\draw[mark=*,mark size=3pt,mark options={color=blue}] plot coordinates {(3,-4)} node[above, blue]{Shirt 3};
\draw (3,4) -- (6,5) ;
\draw[mark=*,mark size=3pt,mark options={color=blue}] plot coordinates {(6,5)} node[above, blue]{Pants 1};
\draw (3,4) -- (6,3) ;
\draw[mark=*,mark size=3pt,mark options={color=blue}] plot coordinates {(6,3)} node[above, blue]{Pants 2};
\draw (3,0) -- (6,1) ;
\draw[mark=*,mark size=3pt,mark options={color=blue}] plot coordinates {(6,1)} node[above, blue]{Pants 1};
\draw (3,0) -- (6,-1) ;
\draw[mark=*,mark size=3pt,mark options={color=blue}] plot coordinates {(6,-1)} node[above, blue]{Pants 2};
\draw (3,-4) -- (6,-3) ;
\draw[mark=*,mark size=3pt,mark options={color=blue}] plot coordinates {(6,-3)} node[above, blue]{Pants 1};
\draw (3,-4) -- (6,-5) ;
\draw[mark=*,mark size=3pt,mark options={color=blue}] plot coordinates {(6,-5)} node[above, blue]{Pants 2};
\end{tikzpicture}
\end{image}
Each node on the far right of the tree diagram represents a different way for Pat to dress. From top to bottom, 
these nodes represent the outfits: Shirt 1 with Pants 1, Shirt 1 with Pants 2, Shirt 2 with Pants 1, 
Shirt 2 with Pants 2, Shirt 3 with Pants 1 and Shirt 3 with Pants 2. 
Since there are 6 of these nodes, Pat can dress in 6 different ways.

\end{example}
